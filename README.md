# Python Intro

This repository documents my self-study journey as I learn how Python works and how it differs from JavaScript.

I’ll be revisiting lessons I originally completed in JavaScript at EDA and rebuilding them in Python where possible.

Not every JavaScript lesson will translate directly to Python, and that is part of the learning process.

## Purpose

The goal of this project is to:

- Learn Python syntax and behavior
- Compare Python with JavaScript
- Document mistakes, discoveries, and progress
- Build a stronger understanding of programming concepts in Python
